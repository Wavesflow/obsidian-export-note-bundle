# obsidian-export-note-bundle

## Description

Easily export Markdown, Excalidraw, Canvas, and `.base` notes together with all their attachments into a clean, self-contained folder or another Obsidian vault. Supports single files, folders, or batch export, and everything just works after export — no broken links.

把 Markdown、Excalidraw、Canvas 和 `.base` 笔记连同所有附件一起导出成一个完整的文件夹或者导出到另外一个 Obsidian 库。支持单文件、文件夹和批量导出，导出后直接可用，不会出现丢附件或链接失效的问题。
